# Ambiente de Git

---

## Sobre el Proyecto
Este proyecto fue creado con el propósito de familiarizarse con el ambiente de las herramientas de **Git** y **GitHub** para los repositorios de proyectos de software, así como el uso de comandos escenciales como `commit`.

---

## Comandos utilizados
A continuación, se muestran los comandos utilizados durante la realización de esta actividad:

| Comando | Función |
|:---:|:---:|
|`git config --global user.name [nombre_usuario_github]`| Asociar nuestro proyecto con nuestro usuario de GitHub. |
|`git git config --global user.email [correo_usuario_github]`| Asociar nuestro proyecto con nuestro correo de GitHub. |
|`git status`| Muestra si el proyecto actual esta configurado como repositorio. |
|`git init`| Inicializa el proyecto actual como un repositorio. |
|`git add .`| Añadir todos los archivos de la carpeta del proyecto al repositorio. |
|`git commit -m "<Tipo>[archivo de referencia]:[descripcion]"`| Creación de una nueva versión/Punto de Guardado. |
|`git log`| Muestra todos los commit creados. |
|`git remote add origin [enlace_repositorio]`| Enlace del repositorio local a un repositorio de GitHub. |
|`git branch -M main`| Se trabaja sobre la rama "main" del repositorio. |
|`git push -u origin main`| Se "empujan" los cambios realizados en el repositorio local al repositorio de GitHub. |
